### Prerequisites

- Node.js 16+ and npm
- Vue3
- Vuex4

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start dev server
npm run dev

# Server runs on http://localhost:5173
```

### Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

### Testing

```bash
# Run tests
npm test

# Run tests in watch mode
npm test -- --watch

# Run tests with UI
npm run test:ui

# Run tests with coverage
npm run test:coverage
```

The project includes comprehensive unit tests for:
- Vuex store modules (horse, race, game)
- Vue components (HorseList, ProgramPanel, ResultsPanel)
- All tests are located in `__tests__` directories
